## usage
### 开发
#### 下载源码
git clone git@github.com:lirongfei123/mock-pro.git
cd mock-pro
#### 数据库
新建一个数据库
将sql目录里面的sql表结构导入，新建表
### 运行
在mock-pro目录下执行 npm install
在mock-pro/be 执行 npm install

### 部署
执行和开发同样的操作
然后执行be下面的start.js文件
比如forever
forever start start.js
